## Report Generator

This repository now includes a small `report_generator.py` script. It demonstrates
how to store calculation results in structured data (a list of dictionaries) and
export them directly to PDF and Word without parsing any intermediate strings.
Run the module directly to generate sample `results.pdf` and `results.docx` files.
